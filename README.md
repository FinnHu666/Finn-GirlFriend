# 给女友的电子点菜菜单 ☁️

基于微信小程序 + 微信云开发搭建的电子点菜菜单，零服务器成本。

## 技术栈

- **前端**：微信小程序（WXML / WXSS / JS）
- **后端**：微信云开发（云函数 + 云数据库）
- **原来**：Node.js → 已迁移至云开发

## 快速开始

### 1. 开通云开发

1. 微信开发者工具打开 `miniprogram` 目录
2. 点击工具栏「云开发」→ 开通 → 创建环境
3. 记下环境 ID，替换 `miniprogram/app.js` 中的 `env` 值

### 2. 创建数据库集合

在云开发控制台 → 数据库 → 创建以下集合：

| 集合 | 说明 |
|------|------|
| `users` | 用户（女友/男友） |
| `categories` | 菜品分类 |
| `tasteTags` | 口味标签 |
| `dishes` | 菜品数据 |
| `orders` | 点菜单/订单 |
| `sessions` | 登录会话 |

### 3. 部署云函数

在开发者工具中，右键 `cloudfunctions/` 下的每个云函数目录 →「上传并部署：云端安装依赖」。

### 4. 导入初始数据

在云开发控制台 → 数据库 → 每个集合点击「导入」→ 选择 `seed/` 目录下对应的 JSON 文件。

### 5. 启动小程序

开发者工具中直接编译运行即可。无需后端服务器！

## 目录结构

```text
miniprogram/          # 微信小程序前端
cloudfunctions/       # 云函数（后端逻辑）
  ├── auth/           # 登录认证
  ├── home/           # 首页聚合
  ├── dishes/         # 菜品查询 & 推荐
  ├── orders/         # 点单 & 查询
  └── admin/          # 男友管理后台
seed/                 # 初始数据（用于导入云数据库）
README.md
```

## 已实现功能

- 首页：专属问候、今日推荐、随机推荐
- 菜单：分类/口味筛选、搜索、加入点菜单
- 详情：口味多选、份量单选、备注
- 点菜单：菜品汇总、心情/饥饿程度、提交订单
- 提交成功：订单摘要与温柔反馈
- 我的：登录状态、角色入口、管理入口
- 登录：账号密码登录，区分女友/男友
- 管理：男友端菜品维护、上下架、推荐
