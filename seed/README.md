# 种子数据说明

本目录中的 JSON 文件用于初始化云数据库。

## 使用方法

1. 打开微信开发者工具 → 云开发控制台 → 数据库
2. 在对应集合中点击「导入」
3. 选择本目录下对应的 JSON 文件

## 文件对应关系

| 文件 | 对应的云数据库集合 |
|------|-------------------|
| `seed.json` | 全部数据（手动拆分或一次性导入） |

## 数据内容

seed.json 包含以下数据：

- **categories**：菜品分类（热菜、凉菜、汤品、主食、饮品、小吃）
- **tasteTags**：口味标签（微辣、麻辣、酸甜、清淡、香酥、浓郁）
- **dishes**：菜品列表（名称、图片、价格、口味、推荐状态）
- **orders**：订单（初始为空，运行后生成）
- **users**：默认用户账号
