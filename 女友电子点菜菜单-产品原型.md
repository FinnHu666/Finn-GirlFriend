# 给女友的电子点菜菜单｜产品原型

版本：V1.0  
日期：2026-07-10  
阶段：低保真原型设计，不涉及代码

---

## 1. 原型范围

本期包括以下页面：

1. 首页
2. 菜单列表页
3. 菜品详情页
4. 今日点菜单页
5. 随机推荐弹窗
6. 提交成功页
7. 菜单管理页

---

## 2. 首页

### 页面目标

提供专属问候、今日推荐及点菜入口，让女友打开后可以立即开始选择。

```text
┌──────────────────────────────────┐
│ 晚上好呀，小宝 🌙                │
│ 今天也要好好吃饭                 │
├──────────────────────────────────┤
│ 今日推荐                         │
│                                  │
│ ┌────────────┐  ┌────────────┐  │
│ │ 番茄牛腩    │  │ 可乐鸡翅    │  │
│ │ 酸甜·热乎  │  │ 香甜·下饭  │  │
│ │ ❤️❤️❤️❤️❤️ │  │ ❤️❤️❤️❤️   │  │
│ └────────────┘  └────────────┘  │
├──────────────────────────────────┤
│ [ 我要点菜 ]                     │
│ [ 随机吃点什么 ]                 │
│ [ 查看今日点菜单 ]               │
├──────────────────────────────────┤
│ 不知道吃什么，就交给我吧 🍰       │
└──────────────────────────────────┘
```

### 交互

- “我要点菜”进入菜单列表页。
- “随机吃点什么”打开推荐弹窗。
- 点击推荐卡片进入菜品详情页。
- “查看今日点菜单”进入订单确认页。

## 3. 菜单列表页

### 页面目标

按分类和口味浏览菜品，快速加入今日菜单。

```text
┌──────────────────────────────────┐
│ ← 今天想吃什么？                 │
├──────────────────────────────────┤
│ [全部] [主食] [荤菜] [素菜]      │
│ [汤品] [甜品] [饮品] [隐藏菜单]  │
│                                  │
│ 口味：[清淡] [微辣] [酸甜]       │
│       [热乎] [下饭]              │
├──────────────────────────────────┤
│ ┌──────────────────────────────┐ │
│ │ 番茄牛腩                     │ │
│ │ 酸甜 / 热乎 / 下饭           │ │
│ │ 约 60 分钟  ❤️❤️❤️❤️❤️        │ │
│ │ [查看详情]       [加入菜单]  │ │
│ └──────────────────────────────┘ │
│                                  │
│ ┌──────────────────────────────┐ │
│ │ 蒜蓉西兰花                   │ │
│ │ 清淡 / 健康 / 快手           │ │
│ │ 约 15 分钟  ❤️❤️❤️            │ │
│ │ [查看详情]       [加入菜单]  │ │
│ └──────────────────────────────┘ │
├──────────────────────────────────┤
│ 已选 2 道菜              [去确认]│
└──────────────────────────────────┘
```

### 交互

- 点击分类或口味标签刷新菜品列表。
- 点击“加入菜单”后显示加入成功提示。
- 底部固定显示已选数量和确认入口。

## 4. 菜品详情页

### 页面目标

查看菜品介绍，选择口味、份量并添加备注。

```text
┌──────────────────────────────────┐
│ ← 番茄牛腩                       │
├──────────────────────────────────┤
│ [       菜品图片占位       ]     │
│                                  │
│ 番茄牛腩                         │
│ 酸酸甜甜、热乎下饭，适合下班后。 │
│                                  │
│ 推荐：❤️❤️❤️❤️❤️                │
│ 制作：约 60 分钟                 │
│ 难度：中等                       │
├──────────────────────────────────┤
│ 口味                             │
│ [清淡] [正常] [重口] [微辣]      │
│ [不要香菜] [多放土豆]            │
│                                  │
│ 份量                             │
│ [小份] [正常] [多一点]           │
│                                  │
│ 备注                             │
│ ┌──────────────────────────────┐ │
│ │ 例如：想多放一点番茄         │ │
│ └──────────────────────────────┘ │
├──────────────────────────────────┤
│ [ 加入今日点菜单 ]               │
└──────────────────────────────────┘
```

### 交互

- 口味支持多选，份量为单选。
- 加入后可选择继续点菜或去确认。
- 重复添加时提示该菜已在菜单中。

## 5. 今日点菜单页

### 页面目标

汇总已选菜品及个性化要求，确认后提交给男友。

```text
┌──────────────────────────────────┐
│ ← 今日点菜单 🧾                  │
├──────────────────────────────────┤
│ 番茄牛腩                         │
│ 正常份 / 微辣 / 多放土豆         │
│                 [修改] [删除]    │
│                                  │
│ 蒜蓉西兰花                       │
│ 小份 / 清淡                      │
│                 [修改] [删除]    │
├──────────────────────────────────┤
│ 饥饿程度                         │
│ [一点点饿] [正常饿] [非常饿]     │
│ [饿到要闹了]                     │
│                                  │
│ 今日心情                         │
│ [开心] [累了] [想被哄] [想吃甜]  │
│                                  │
│ 期望用餐时间                     │
│ [18:30] [19:00] [19:30] [自定义] │
│                                  │
│ 给男友的备注                     │
│ ┌──────────────────────────────┐ │
│ │ 今天想吃热乎一点的           │ │
│ └──────────────────────────────┘ │
├──────────────────────────────────┤
│ 预计制作时间：约 70 分钟         │
│ [ 提交给男友 ]                   │
└──────────────────────────────────┘
```

### 交互

- 支持修改或删除已选菜品。
- 未选择菜品时提交按钮不可用。
- 提交前显示二次确认弹窗。

## 9. 页面流转

```text
首页
├─ 我要点菜 → 菜单列表 → 菜品详情 → 今日点菜单
├─ 随机推荐 → 推荐套餐 ───────────→ 今日点菜单
└─ 查看订单 ─────────────────────→ 今日点菜单
                                      ↓
                                  提交成功页
```

## 10. 页面状态

### 空菜单

```text
今日点菜单还是空的。
先去选一道想吃的菜吧 🍚
[ 去点菜 ]
```

### 添加成功

```text
已加入今日点菜单 ❤️
要不要再选一道汤或者甜品？
[继续点菜] [去确认]
```

### 删除确认

```text
要把这道菜从今日菜单里移除吗？
[再想想] [确认删除]
```

## 11. 视觉方向

| 项目 | 设计建议 |
|---|---|
| 主色 | 奶油粉、蜜桃粉、浅米白 |
| 辅色 | 柠檬黄、薄荷绿、浅棕色 |
| 卡片 | 大圆角、轻阴影、充足留白 |
| 图标 | 线性图标配少量 Emoji |
| 字体 | 圆润、亲切、清晰 |
| 动效 | 柔和弹出、爱心反馈、轻微过渡 |

原型应优先适配手机竖屏，同时保证未来可以扩展为电脑网页。
