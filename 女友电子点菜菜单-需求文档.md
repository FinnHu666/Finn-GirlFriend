## 1. 项目背景

希望为女友设计一份专属的“电子点菜菜单”。它不是普通餐厅菜单，而是一个带有仪式感、偏温柔可爱风格的小型点餐产品：女友可以像在餐厅一样浏览今日菜单、选择想吃的菜、备注口味偏好，并生成一份“今日投喂订单”。

该产品可以用于日常做饭、周末约会、纪念日惊喜，也可以逐步扩展为情侣之间的生活小工具。

---

## 2. 产品目标

### 2.1 核心目标

1. 让女友可以轻松选择今天想吃什么。
2. 降低“今天吃什么”的沟通成本。
3. 增加生活仪式感和互动感。
4. 帮助男友提前准备食材、安排做饭顺序。

### 2.2 非目标

当前阶段不做以下内容：

- 不涉及真实支付。
- 不涉及外卖平台对接。
- 不涉及多人餐厅排队、库存、后厨系统。
- 不涉及复杂账号体系。
- 不涉及代码开发。

## 6. 功能需求

## 6.1 首页

### 功能描述

首页用于承载欢迎语、今日推荐和核心操作入口。

### 页面元素

| 模块 | 内容 |
|---|---|
| 顶部问候 | 根据时间展示“早安”“午安”“晚上想吃点什么呀？” |
| 女友昵称 | 可配置，如“小宝”“宝宝”“老婆大人” |
| 今日推荐 | 展示 1-3 个推荐菜品 |
| 快捷按钮 | 我要点菜、随机推荐、查看今日点菜单 |
| 温柔文案 | 例如“今天也要好好吃饭呀” |

### 交互规则

- 点击“我要点菜”进入菜单页。
- 点击“随机吃点什么”弹出推荐菜品。
- 点击推荐菜品进入菜品详情页。

---

## 6.2 菜单页

### 功能描述

展示所有可选菜品，支持按分类、口味、制作时间进行筛选。

### 分类建议

| 分类 | 示例 |
|---|---|
| 主食 | 米饭、炒饭、面条、饺子、粥 |
| 荤菜 | 可乐鸡翅、红烧肉、土豆牛腩、番茄牛腩 |
| 素菜 | 蒜蓉西兰花、酸辣土豆丝、手撕包菜 |
| 汤品 | 番茄蛋汤、玉米排骨汤、紫菜蛋花汤 |
| 甜品 | 双皮奶、芋圆、布丁、小蛋糕 |
| 饮品 | 奶茶、柠檬茶、热牛奶、蜂蜜水 |
| 隐藏菜单 | 男友特制、纪念日限定、随机惊喜 |

### 菜品卡片字段

| 字段 | 说明 |
|---|---|
| 菜名 | 菜品名称 |
| 图片/插画 | 可先用占位图描述 |
| 口味标签 | 微辣、清淡、酸甜、热乎、下饭 |
| 制作时间 | 例如 15 分钟、30 分钟、60 分钟 |
| 难度 | 简单、中等、复杂 |
| 推荐指数 | 1-5 颗爱心 |
| 女友喜爱度 | 可记录历史偏好 |

---

## 6.3 菜品详情页

### 功能描述

展示某一道菜的详细信息，并允许女友选择口味和备注。

### 页面元素

| 模块 | 内容 |
|---|---|
| 菜品名称 | 如“番茄牛腩” |
| 菜品说明 | 简短介绍 |
| 推荐理由 | 如“酸酸甜甜，很适合今天下班后吃” |
| 口味选择 | 清淡、正常、重口、微辣、不要香菜等 |
| 份量选择 | 小份、正常、多一点 |
| 备注 | 自由输入，如“想多放土豆” |
| 加入按钮 | 加入今日点菜单 |

### 交互规则

- 同一道菜可重复选择，但默认提示“已经在今日菜单里啦”。
- 必填项：菜品名称、份量。
- 选填项：口味、备注。

---

## 6.4 今日点菜单

### 功能描述

汇总女友已选择的菜品，允许最终确认和提交。

### 页面元素

| 模块 | 内容 |
|---|---|
| 已选菜品列表 | 展示菜名、份量、口味、备注 |
| 总制作时间估算 | 根据菜品制作时间估算 |
| 饥饿程度 | 一点点饿、正常饿、非常饿、饿到要闹了 |
| 期望用餐时间 | 如 18:30、19:00 |
| 今日心情 | 开心、累了、想被哄、想吃甜的 |
| 提交按钮 | 提交给男友 |

### 交互规则

- 若未选择菜品，提交按钮不可用。
- 提交前展示确认弹窗。
- 提交后进入成功页。

---

## 6.5 随机推荐

### 功能描述

当女友不知道吃什么时，由系统根据分类、心情、天气、历史喜好随机推荐。

### 推荐维度

| 维度 | 示例 |
|---|---|
| 心情 | 累了、开心、想撒娇、没胃口 |
| 饥饿程度 | 轻食、正常餐、大餐 |
| 天气 | 冷天推荐热汤，热天推荐清爽饮品 |
| 历史偏好 | 优先推荐高喜爱度菜品 |

### 推荐结果

- 推荐 1 道主菜。
- 可搭配 1 道素菜/汤品/饮品。
- 提供“换一个”按钮。

---

## 6.6 菜单管理配置

### 功能描述

由男友维护菜品内容，不面向女友重点展示。

### 维护字段

| 字段 | 是否必填 | 说明 |
|---|---|---|
| 菜名 | 是 | 菜品名称 |
| 分类 | 是 | 主食/荤菜/素菜等 |
| 描述 | 是 | 简短介绍 |
| 制作时间 | 是 | 分钟 |
| 难度 | 否 | 简单/中等/复杂 |
| 口味标签 | 否 | 多选 |
| 是否推荐 | 否 | 是否进入今日推荐 |
| 是否隐藏菜单 | 否 | 是否只在特殊入口展示 |
| 女友喜爱度 | 否 | 1-5 |

## 7. 非功能需求

| 类型 | 要求 |
|---|---|
| 视觉风格 | 温柔、可爱、清爽，不要过度商业化 |
| 使用门槛 | 女友无需学习，打开即会用 |
| 响应速度 | 页面切换轻快，操作不拖沓 |
| 可维护性 | 菜品、分类、文案应方便后续调整 |
| 私密性 | 仅用于个人生活场景，不公开展示个人信息 |
| 扩展性 | 后续可扩展纪念日菜单、历史订单、偏好分析 |

## 8. 内容设计

### 8.1 文案风格

建议文案：

- “今天想吃点什么呀？”
- “宝宝今天要好好吃饭。”
- “已加入今日投喂菜单。”
- “男友正在接收你的点菜信号。”
- “不要纠结啦，我帮你推荐一个。”

避免文案：

- 过于正式的餐饮系统用语。
- 命令式表达。
- 冰冷的错误提示。

### 8.2 推荐菜品初始清单

| 分类 | 菜品 |
|---|---|
| 主食 | 蛋炒饭、番茄鸡蛋面、牛肉面、鲜虾馄饨 |
| 荤菜 | 可乐鸡翅、番茄牛腩、土豆烧牛肉、糖醋里脊 |
| 素菜 | 蒜蓉西兰花、酸辣土豆丝、清炒娃娃菜、手撕包菜 |
| 汤品 | 番茄蛋汤、玉米排骨汤、紫菜蛋花汤 |
| 甜品 | 焦糖布丁、芋圆、双皮奶、小蛋糕 |
| 饮品 | 柠檬茶、热牛奶、蜂蜜水、奶茶 |
| 隐藏菜单 | 男友特制煎蛋、纪念日限定套餐、哄哄套餐 |

## 9. 用户流程

### 9.1 正常点菜流程

```text
打开首页
  ↓
查看今日问候和推荐
  ↓
进入菜单页
  ↓
筛选分类/浏览菜品
  ↓
进入菜品详情
  ↓
选择口味、份量、备注
  ↓
加入今日点菜单
  ↓
确认今日点菜单
  ↓
提交给男友
  ↓
看到提交成功与等待投喂提示
```

### 9.2 不知道吃什么流程

```text
打开首页
  ↓
点击“随机吃点什么”
  ↓
选择当前心情/饥饿程度
  ↓
系统推荐套餐
  ↓
女友选择“就这个”或“换一个”
  ↓
加入今日点菜单
```
